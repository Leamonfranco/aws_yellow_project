Nuestro Proyecto

Despliegue de una aplicación web con alta disponibilidad en AWS garantizando que permanezca accesible incluso si una zona de disponibilidad de AWS falla.

Objetivos Principales

  • Implementar alta disponibilidad mediante múltiples zonas de disponibilidad
  • Asegurar el acceso a la web exclusivamente a través del balanceador de carga
  • Mantener la continuidad del servicio ante fallos de instancias
  • Implementar escalado automático basado en la demanda
  • Desarrollar competencias técnicas en todo el equipo

Resultados Esperados

  • Diseño e implementación de arquitectura cloud robusta
  • Distribución de carga efectiva entre múltiples instancias
  • Sistema de auto-escalado funcional
  • Equipo capacitado para mantener y explicar la infraestructura

Arquitectura del proyecto

Tecnologías Utilizadas

  • EC2, Balanceador de carga, Grupos de seguridad
  • Github, HTML, CSS, Javascript

Competencias Desarrolladas

  • Desarrollo y despliegue de web estática
  • Configuración de servidores web en instancias EC2
  • Implementación de sistemas de auto-escalado
  • Gestión de balanceadores de carga
  • Configuración de seguridad en la nube
